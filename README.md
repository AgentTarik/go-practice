# Practice

Go algorithm practice. Each question lives in its own package under `questions/`.

## Running

```bash
# Run a specific question
go run ./questions/01-islands/
go run ./questions/02-two-sum/
go run ./questions/03-merge-intervals/
go run ./questions/04-longest-substring/
go run ./questions/05-binary-search/
go run ./questions/06-valid-parentheses/

# Build all packages
go build ./questions/...
```
